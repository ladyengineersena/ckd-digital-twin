# ETHICS / Data Governance

- This repo contains only synthetic demo data. Do NOT use real patient data without IRB approval.

- Models are research prototypes and not validated for clinical decisions.

- If real data will be used: obtain informed consent, IRB approval, and Data Transfer Agreement (DTA). Pseudonymize data before processing.

- Keep audit logs and restrict data access in production.
